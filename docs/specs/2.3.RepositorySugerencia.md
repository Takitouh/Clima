# 2.3 Repository Sugerencia - User Feedback Management

## Overview

Implementation of the user feedback/suggestions repository layer. Handles CRUD operations for
storing user feedback messages in the local database. This repository manages user-provided
suggestions about the weather app functionality.

**Location:** `data/repository/impl/RepositorySugerencia.kt`

**Interface:** `data/repository/IRepositorySugerencia.kt`

**Status:** ✅ COMPLETE

---

## Entity: Sugerencia (User Feedback)

```kotlin
@Entity(tableName = "sugerencia")
data class Sugerencia(
    @PrimaryKey(autoGenerate = true)
    var id: Int = 0,
    @ColumnInfo(name = "nombre_completo")
    var nombreCompleto: String,      // Full name of user providing feedback
    var mensaje: String,              // Feedback message content
    @ColumnInfo(name = "fecha_enviado")
    var fechaEnviado: String         // Date/time when feedback was submitted
)
```

**Purpose:**

- Stores user-provided suggestions and feedback
- Maintains user contact information for follow-up
- Tracks when feedback was submitted
- Enables feedback analytics and improvement tracking

---

## DAO: DaoSugerencia

```kotlin
@Dao
interface DaoSugerencia {
    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertar(sugerencia: Sugerencia): Long

    @Query("SELECT * FROM sugerencia")
    suspend fun obtenerTodos(): List<Sugerencia>

    @Update
    suspend fun actualizar(sugerencia: Sugerencia): Int

    @Query("DELETE FROM sugerencia WHERE id = :id")
    suspend fun eliminar(id: Int): Int
}
```

**Methods:**

- `insertar()`: Adds new feedback (replace on conflict)
- `obtenerTodos()`: Retrieves all feedback records
- `actualizar()`: Updates existing feedback
- `eliminar()`: Deletes feedback by ID

## Repository Interface: IRepositorySugerencia

### Method: insert()

**Creates a new feedback record in the database.**

```kotlin
suspend fun insert(sugerencia: Sugerencia): Long
```

**Parameters:**

- `sugerencia`: Sugerencia object containing full name, message, and date

**Returns:**

- `Long`: ID of newly inserted record

**Error Handling:**

- Throws `Exception` if database operation fails

**Use Cases:**

- Store new user feedback immediately after submission
- Create audit trail of user suggestions

**Example:**

```kotlin
val feedback = Sugerencia(
    nombreCompleto = "John Doe",
    mensaje = "Please add hourly weather forecast feature",
    fechaEnviado = "2026-05-01 10:30:00"
)

val feedbackId = repositorySugerencia.insert(feedback)
Log.d("Feedback", "Feedback stored with ID: $feedbackId")
```

---

### Method: obtainAll()

**Retrieves all feedback records from database.**

```kotlin
suspend fun obtainAll(): List<Sugerencia>
```

**Returns:**

- `List<Sugerencia>`: All feedback records, empty list if none exist

**Error Handling:**

- Throws `Exception` if database operation fails

**Use Cases:**

- Display feedback history in admin panel
- Analyze user suggestions for feature prioritization
- Generate feedback reports
- Backup user feedback data

**Example:**

```kotlin
val allFeedback = repositorySugerencia.obtainAll()
Log.d("Feedback", "Retrieved ${allFeedback.size} feedback records")

allFeedback.forEach { suggestion ->
    println("${suggestion.nombreCompleto}: ${suggestion.mensaje}")
}
```

---

### Method: update()

**Modifies an existing feedback record.**

```kotlin
suspend fun update(sugerencia: Sugerencia): Int
```

**Parameters:**

- `sugerencia`: Sugerencia object with updated data (must have ID set)

**Returns:**

- `Int`: Number of records updated (0 or 1)

**Error Handling:**

- Throws `Exception` if database operation fails

**Important:**

- Sugerencia must have valid ID from database
- Returns 0 if ID doesn't exist

**Use Cases:**

- Correct mistaken feedback entries
- Mark feedback as reviewed or processed
- Update feedback status

**Example:**

```kotlin
val existingFeedback = repositorySugerencia.obtainAll().firstOrNull()
if (existingFeedback != null) {
    existingFeedback.mensaje = "Updated feedback message"
    val updatedCount = repositorySugerencia.update(existingFeedback)
    Log.d("Feedback", "Updated $updatedCount record(s)")
}
```

---

### Method: delete()

**Removes a feedback record from database.**

```kotlin
suspend fun delete(id: Int): Int
```

**Parameters:**

- `id`: Primary key of feedback record to delete

**Returns:**

- `Int`: Number of records deleted (0 or 1)

**Error Handling:**

- Throws `Exception` if database operation fails

**Use Cases:**

- Remove duplicate feedback entries
- Delete spam or inappropriate feedback
- Manage database storage

**Example:**

```kotlin
val deletedCount = repositorySugerencia.delete(feedbackId)

```

## Implementation Details

### Hilt Integration

The RepositorySugerencia is injected via Hilt:

```kotlin
class RepositorySugerencia @Inject constructor(
    private val daoSugerencia: DaoSugerencia
) : IRepositorySugerencia
```

**Dependencies:**

- `DaoSugerencia` - Database access object

### Logging

All operations logged with TAG = "RepositorySugerencia":

- **DEBUG**: Operation start, success, record count
- **ERROR**: Operation failures with exception details

### Coroutine Context

All database operations run on `Dispatchers.IO` for non-blocking database access.

## Error Handling

| Error Type                   | Cause                  | Response         | Recovery                    |
|------------------------------|------------------------|------------------|-----------------------------|
| Database constraint violated | Duplicate/invalid data | Exception thrown | Validate data before insert |
| Database operation timeout   | DB locked or slow      | Exception thrown | Retry with backoff          |
| Disk space full              | Storage exhausted      | Exception thrown | Cleanup or request storage  |

## Data Flow Example

### Complete Feedback Submission Flow

```kotlin
// 1. Create feedback object
val feedback = Sugerencia(
    nombreCompleto = "Jane Smith",
    mensaje = "Add push notifications for severe weather alerts",
    fechaEnviado = TimeUtil.getCurrentUtcDateTime()
)

// 2. Store to database via repository
try {
    val feedbackId = repositorySugerencia.insert(feedback)
    Log.d("Feedback", "Successfully stored feedback with ID: $feedbackId")

    // 3. Optional: Show confirmation to user
    showSuccessMessage("Thank you for your feedback!")

} catch (e: Exception) {
    Log.e("Feedback", "Failed to store feedback: ${e.message}")
    showErrorMessage("Could not save feedback. Please try again.")
}
```
